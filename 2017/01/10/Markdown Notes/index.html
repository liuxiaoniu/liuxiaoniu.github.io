Markdown Notes

发表于 2017-01-10

This is a smart aticle about how to write in Markdown for this blog,just like CheatSheet for Mac.

Primer

  1. 标题: #的数目后空格(1~6)
  2. 斜体 / 粗体: *夹 / **夹
  3. 划分段落: 宽-空白行; 窄-2空格后回车(块注释里同样适用)
  4. 块注释: >(5空格后会框起来)(可嵌套)
  5. 有序列表 / 无序列表: number.空格(可嵌套) / -空格
  6. 图片: ![替换文本](链接 “tips文本”) (用标签可指定图片大小)
  7. 网址: [要加链接的内容](链接 “tips文本”)
  8. 邮箱 / 网址本身链接: <内容>
  9. 代码: 小段-`夹; 大段-4空格
  10. 水平线: 3个-
  11. 转义: 用\来转义表示文本中的Markdown符号

Advance

  1. 删除: ~~夹
  2. 表格: 详见下面[Advance Test Below]
  3. 流程图: 待更新

Attention:

  1. 文首输入tags添加标签，categories添加分类(有层次)
  2. 可兼容html标签
  3. 不同Markdown环境可能有细微差异，记得预览后微调(如:注意前后空行)
  4. 有些高级功能属于衍生功能, 可能不支持
