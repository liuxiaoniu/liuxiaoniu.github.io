学习“Markdown”标记语言

发表于 2017-01-10

This is a smart aticle about how to write in Markdown for this blog,just like CheatSheet for Mac(cope).

入门指南

  1. 标题: #的数目后空格（1~6阶）
  2. 斜体 / 粗体: *夹起来 / ** 夹
  3. 划分段落: 宽-空白行；窄–空白格后回车（块注释同样适用）
  4. 块注释: >（5空格会框起来）（可嵌套）
  5. 有序列表 /无序列表: 数字.空格（可嵌套）/ -空格
  6. 图片: ！[替换文本](链接“tips”文本) （用img>标签可指定图片大小）
  7. 网址： [要加链接的内容](链接 “tips文本”)
  8. 邮箱 / 网址本身链接: <内容>
  9. 代码: 小段-`夹; 大段-4空格
